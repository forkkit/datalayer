# LDAP Helm Chart

[OpenLDAP Helm Chart](https://github.com/helm/charts/tree/master/stable/openldap).

```bash
export RELEASE=ldap
export NAMESPACE=ldap
helm upgrade \
  --install $RELEASE \
  stable/openldap \
  --namespace $NAMESPACE \
  --timeout 99999 \
  --set service.type=NodePort \
  --values ldap.yaml
```
